# Simple Calculator (Python)

A basic command-line calculator built using Python.  
This mini project performs addition, subtraction, multiplication, and division.
## Features
- Easy-to-use command-line interface
- Performs 4 basic operations
- Input validation included
- Beginner-friendly code structure
## Requirements
- Python 3.x
##  Installation & Usage
### 1. Clone the Repository
git clone https://github.com/Mzlh2/cs_assignment4.git
### 2. Run the Program
python calculator.py
